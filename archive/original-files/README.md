# Archived Original Files

This local folder contains historical development material retained for comparison. Git ignores every archived item except this warning, so the historical material is not part of the public release.

> **Warning:** Do not execute or publish archived files without reviewing them first. Some contain historical hardcoded network ranges, hostnames, scan timestamps, contributor metadata, automatic `/24` scanning behavior, optional dependency imports, and misleading Online/Offline terminology. They are retained only for educational comparison and project history.

Use the cleaned files in the repository root and `examples/` for learning or demonstrations.
