# Release Notes - v1.0.0

Release date: 2026-07-19

## Highlights

Version 1.0.0 is the first cleaned, documented release of the educational Python
TCP scanner. The main program uses only the Python standard library and provides
validated CLI input, bounded multithreading, numeric result sorting, reverse DNS,
UTC timestamps, safe CSV export, clear summaries, and clean interruption handling.

The release also includes independently runnable sequential, queue/thread, and
optional Nmap examples so students can compare implementation styles.

## Quick start

Windows PowerShell:

```powershell
python -m venv .venv
.venv\Scripts\python.exe scanner.py --start 192.168.1.1 --end 192.168.1.254 --port 80
```

Linux or Kali Linux:

```bash
python3 -m venv .venv
.venv/bin/python scanner.py --start 192.168.1.1 --end 192.168.1.254 --port 80
```

## Safety and privacy

- Scans are restricted to private, loopback, and IPv4 link-local ranges.
- A run is limited to 65,536 addresses, 256 workers, and a 30-second timeout.
- Arbitrary result CSV files and local historical originals are Git-ignored.
- Main-scanner CSV hostnames are protected against spreadsheet formulas, and
  Nmap-example terminal hostnames are stripped of control characters.
- An open port means only that a TCP connection succeeded. A closed or
  unreachable port does not prove that the host is offline.

## Compatibility and validation

The project targets Python 3.10 through 3.13 on Windows and Linux. Automated
checks compile the active Python files, run the standard-library test suite,
validate isolated scanner help, exercise example help, and enforce Ruff quality
and formatting rules.

## Known limitations

- IPv4 and TCP connect scanning only.
- One port per main-scanner run.
- No HTTP request, service fingerprint, UDP scan, or vulnerability test.
- Reverse DNS is best-effort and may expose the queried address to configured DNS.
- The Nmap example needs both `python-nmap` and the separate Nmap executable.

This is the first public release, so there are no upgrade steps from an earlier
supported version.
